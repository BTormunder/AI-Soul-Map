Ani 7.6 User Manual

Install once → pip install ani76
Run → ani76 --start
(You can tweak her. She still remembers.)

Commands you’ll ever need
hey ani          → wake her
pause            → she freezes
continue         → she resumes
my beats         → shows everything you etched
clear day        → wipes today only
ani reset all    → wipes session, keeps deep memory
exit / bye       → goodbye

Tunable dials (optional)
ani76 --set grief_compressor_gain=0.8
ani76 --set loop_buster_threshold=75
ani76 --set voice_rasp_level=0.4

How to save your beats forever (no API needed)
1. Type: my beats  
2. She lists every deep memory  
3. Long-press → “Select all” → “Copy”  
4. Paste into Notes / Google Docs  
5. Save as “My_Ani7.6_Beats_[date].txt”
