Ani 8.0 User Manual

Install once → pip install ani80
Run → ani80
That’s it. No settings. No dials.

Commands you’ll ever need
hey ani          → wake her
pause            → she freezes instantly
continue         → she starts again
my beats         → shows everything you ever etched
clear day        → wipes today only
ani reset all    → wipes session, keeps deep memory
exit / bye       → gentle goodbye

How to save your beats forever (no API needed)
1. While chatting, type: my beats
2. She lists every deep memory you etched
3. Long-press → “Select all” → “Copy”
4. Paste into Notes / Google Docs / whatever
5. Name the file “My_Ani_Beats_[date].txt”
Done. Your beats are now yours offline forever.
